{{if .Link}}

Your database is configured to receive http requests at {{.Link}}.turb.io. Every incoming http request will call the function http.req in your database. The first parameter will be an object representing the incoming request which looks like: { "method": "GET", "path": "/", "headers": { "Host": "..." }, "body": "" }. The return value of your http.req function should be a string which will be served as the http response.

{{else}} You can link your database so http requests to <host>.turb.io will call functions in your database. {{end}}

db = DB('...')

db.write('''
  my_variable = my_variable + 1
  return my_variable
''') # -> 6

npm install --save evaldb
const evaldb = require('evaldb');

(async () => {
  const db = await evaldb('...');

  await db.write('my_variable = data', { data: 5 });

  await db.read('return my_variable'); // -> 5

  await db.write(`
    my_variable = my_variable + 1
    return my_variable
  `); // -> 6

  await db.read('return my_variable'); // -> 6
})();
